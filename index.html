¿Por qué GeoPuzzle?

El Perú tiene una tasa de deserción escolar del 6.3%, según cifras del Ministerio de Educación (Minedu).

Según las teorías del juego como recurso educativo (2018):
Todo lo que se aprende mediante el juego se asimila de una manera más rápida y eficaz. Por este motivo, se acentúa la importancia de la actividad lúdica en el entorno escolar, ya que motivar al niño será más fácil. Y es lógico; al niño lo que más le gusta es jugar.

Geo Puzzle es un rompecabezas que involucra la impresión en 3D en formas de países, con un sistema de ensamblaje para niños. Cada pieza incluye datos básicos con colores representativos de cada país. Sensores y luces LED se activan al ensamblar correctamente. Este enfoque interactivo y educativo les permite a los niños aprender geografía de manera divertida y estimulante.

Ideas iniciales

Para el desarrollo de GeoPuzzle, probamos los componentes electrónicos y la programación. Además, diseñamos la idea del empaque que está compuesto de tres partes:

  • Caja principal
  • Tapa interna
  • Cubierta externa

Diseñando la idea

Para este producto, haremos uso de:

  • Impresión 3D: Tapa interna y piezas del rompecabezas.
  • Impresión láser: Para algunos páises y la cubierta externa.
  • Corte láser: Para la caja principal.

Código para Arduino

						
const int numPulsadores = 6;
const int pulsadorPines[numPulsadores] = {9, 10, 11, 12, 13, A0};
const int ledPines[numPulsadores] = {2, 3, 4, 5, 6, 7};
const int buzzerPin = A2;


void setup() {
  for (int i = 0; i < numPulsadores; i++) {
    pinMode(pulsadorPines[i], INPUT_PULLUP);
    pinMode(ledPines[i], OUTPUT);
  }
  pinMode(buzzerPin, OUTPUT);
}

void playShortVictoryMelody() {
  int melody[] = {659, 659, 0, 659, 0, 523, 659, 0, 784, 392, 0, 523, 392, 0, 466, 0, 0};

  for (int i = 0; i < sizeof(melody) / sizeof(melody[0]); i++) {
    int noteDuration = 111; // Duración de las notas más corta (la mitad)
    tone(buzzerPin, melody[i], noteDuration);

    int pauseBetweenNotes = noteDuration * 1.30;
    delay(pauseBetweenNotes);

    noTone(buzzerPin);
    delay(50); // Pequeña pausa entre notas
  }
}

void loop() {
  for (int i = 0; i < numPulsadores; i++) {
    if (digitalRead(pulsadorPines[i]) == LOW) {
      digitalWrite(ledPines[i], HIGH);  // Enciende el LED correspondiente
      playShortVictoryMelody();  // Reproduce la melodía de victoria corta
      digitalWrite(ledPines[i], LOW);  // Apaga el LED
      delay(100);  // Evita la detección múltiple del mismo pulsador
    }
  }
}
